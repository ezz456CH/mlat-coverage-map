MLAT sync stats for Region 0

This table shows the current clock synchronization state between all receivers. It will automatically update about every 1 minutes.

Each receiver line has the following data:

  • Receiver Name
  • Number of peers this receiver is synchronized with
  • 3 rows that contain:
    • Number of synchronizations done in the last ~1 min ( > 10 = green )
    • Sync error in μsec - CHECK YOUR LOCATION if you have lots of red here ( < 2 = green )
    • PPM offset - these are relative values, but if you have lots of red, check the ppm setting for readsb or dump1090 ( < 50 = green )
